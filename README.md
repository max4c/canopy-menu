Download the "Canopy.zip" file from the [latest release](https://github.com/max4c/canopy-menu/releases/latest) and move the unzipped app into your `Applications` folder.

## Features

### Menu bar item management

- Hide and show menu bar items
- "Always-hidden" menu bar section
- Show hidden items on hover, click, or scroll
- Automatically rehide menu bar items
- Drag and drop to arrange individual items
- Display hidden items in a separate bar (great for MacBooks with the notch)
- Search menu bar items
- Custom item spacing

### Menu bar appearance

- Menu bar tint (solid and gradient)
- Menu bar shadow and border
- Custom menu bar shapes (rounded and/or split)

### Hotkeys

- Toggle individual menu bar sections
- Show the search panel
- Enable/disable the Ice Bar
- Show/hide section divider icons
- Toggle application menus

### Other

- Launch at login
- Automatic updates

## System Requirements

macOS 14 (Sonoma) or later.
